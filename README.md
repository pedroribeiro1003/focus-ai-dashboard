# 🚀 FocusAI - Dashboard de Produtividade

Aplicação web full stack para gerenciamento de tarefas com visualização de produtividade em gráfico.

## 🧠 Funcionalidades

- ✅ Adicionar tarefas
- 📊 Visualizar produtividade em gráfico
- 🎯 Interface moderna e responsiva
- ⚡ Atualização em tempo real

## 🛠️ Tecnologias utilizadas

### Frontend
- React.js
- Chart.js

### Backend
- Python
- FastAPI
